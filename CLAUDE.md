# SmartDisplay (Mural)

PWA para iPad que exibe relógio, clima, fotos e câmeras ao vivo.

## Stack
- HTML/CSS/JS puro (sem frameworks)
- PWA com service worker (`sw.js`)
- Manifesto em `manifest.json`

## Estrutura
- `index.html` — entrada principal
- `js/app.js` — lógica da aplicação
- `css/` — estilos
- `icons/` — ícones da PWA
- `smartdisplay/` — assets adicionais

## Contexto
Este projeto é 100% independente do projeto checkvendas. Não compartilha banco de dados, autenticação nem código com checkvendas.
